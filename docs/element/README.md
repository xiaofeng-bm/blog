# 前言
作为一个前端开发，经常用`Vue`、 `element-ui`等库。用了块一年了吧，也仅仅是停留在会用的基础上，最近随着业务越来越多，渐渐的发现有时候`ui`库提供的一些方法、属性等不能满足开发需求，于是时不时的就去看看`element-ui`的源码、有时候不得已还会将组件代码`copy`一份出来改。  

[该项目源码地址](https://github.com/xiaofeng-bm/element-ui)
